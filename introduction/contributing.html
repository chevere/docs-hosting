# Contributing

# Code

You can help with development by providing your time and care. Chevere is open for any issue (opens new window) and pull request (opens new window), bug reporting, testing, translating, documenting, design, etc.

# Project

You can help to build a bigger project by doing the following:

  • Engaging in GitHub discussions (opens new window)
  • Mentioning Chevere to your friends or colleagues
  • Writing about it (guides, blog posts, etc.)
  • Sharing your experiences and feedback on social media
  • Participating in community events and meetups

# Contact

You can contact the developer at rodolfo@chevere.org. For general inquiries, you can also reach out via the project's GitHub discussions (opens new window).
